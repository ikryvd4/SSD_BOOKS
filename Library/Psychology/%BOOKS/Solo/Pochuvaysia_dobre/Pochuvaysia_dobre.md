
### 1. Емоційне мислення

```
Якщо почуваєшся погано — значить так і є.  
«Я тривожусь — значить, щось не так»
```

### 2. Так має бути 

```
Нав'язуєш собі або іншим жорсткі правила.
«Я повинен завжди бути продуктивним!» 
«Вони мають поводитися ідеально!»
```

### 3. Ментальні ярлики

```
Клеймиш себе або інших одним негативним словом.  
«Я — невдаха», «Він — ідіот».
```

### 4. Читання думок 📌 Зашвидкі висновки

```
Впевнений, що знаєш, що інші думають про тебе (і це точно щось погане).  
«Вони думають, що я дурень…»
Передбачення того що все буде погано
```

### 5. Негативне фільтрування 📌

```
Зосереджуєшся тільки на поганому.  
«Всі похвалили, але один критикнув — я невдаха!»
```

### 6. Ігнорування позитивного 📌

```
Не враховуєш позитивні моменти або вважаєш, що вони не рахуються.  
💬 «Вони мені просто лестили…»
```


### 7. Персоналізація 📌

```txt
 Все погане — через мене.  
 💬 «Вона засмучена? Напевно, я щось не так сказав…»_
```

### 8. Все або нічого 📌

```txt
Якщо не ідеально — значить провал.  
💬 «Я не зробив усе з ToDo — значить я ледар!»
```

### 9. Перебільшення або `Переменшення`

```
Очікуєш найгірше у будь-якій ситуації.  
💬 «Якщо я провалю цю презентацію — мене звільнять і я залишусь без дому!»
```

### 10. Надмірне узагальнення 

```
Якщо сталося щось негативне це призводить до сноуболу - негативний шар
```
